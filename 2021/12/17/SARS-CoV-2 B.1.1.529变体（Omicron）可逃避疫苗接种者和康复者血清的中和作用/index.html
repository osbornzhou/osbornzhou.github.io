2021-12-17

SARS-CoV-2 B.1.1.529变体（Omicron）可逃避疫苗接种者和康复者血清的中和作用

摘要

最近，严重急性呼吸系统综合症冠状病毒2（SARS-CoV-2）变体B.1.1.529（Omicron）已被报道出来。

在此，我们分析了来自已从感染新冠病毒中康复的个体或接种过新冠疫苗的个体血清对新的B.1.1.529变体的中和抗体效价，并使用具有复制能力的 SARS-CoV-2 变体来将它们与针对其他关注的变体（B.1.1.7、B.1.351、B.1617.2）的效价进行比较。

我们发现，来自接种者的血清对B.1.1.529变体的中和程度远远低于所分析的任何其他变体。对B.1.1.529的中和能力在来自超级免疫个体（感染后接种疫苗者或接种疫苗后感染者）的血清中保持得最好。

在SARS-CoV-2大流行期间，出现了多个新的病毒变体，从而使病毒变得更具传染性。然而，之前我们没有观察到这些变体存在有效的免疫逃逸，因此疫苗仍然具有有效性。最近，B.1.1.529变体（Omicron）被报道出来，然后世界卫生组织（WHO）于2021年11月26日将其列为关注变体（VOC）。

B.1.1.529的特点是大量的突变。它在尖峰（S）蛋白中有26-32个变化。 由于这些突变中有许多位于已知的参与免疫逃逸的区域，于是我们研究了来自接种疫苗或康复期个体的血清是否能中和这种新的“关注变体”。我们观察到B.1.1.529比以前的变体更容易引起再感染，而这确实表明了某种程度上的免疫逃逸现象。

我们从曾感染过B.1.1.7（n=10）或B.1.351（n=8）或B.1.617.2（n=7）的患者，或者从用Spikevax（Moderna，n=10）、ChAdOx1（Vaxzevria，AstraZeneca）（n=10）或BNT162b2（Comirnaty，Pfizer/BioNTech）（n=20）进行过同源初始免疫/加强免疫疫苗接种后的个体中挑选血清并在异源ChAdOx1初始免疫/BNT162b2加强免疫疫苗接种后（n=20）（见补充附录的表S1和S2）。如前所述，对于所有血清，我们使用了具有复制能力的 SARS-CoV-2 病毒的病灶形成试验确定了针对 B.1.1.7、B.1.351、B.1.617.2 和 B.1.1.529 变体的中和抗体的效价。此外，我们使用了超级免疫者的血清（感染后接种疫苗的人数为5人，接种疫苗后感染的人数为5人，见补充附录中的表S3），分析了针对B.1.617.2和B.1.1.529的中和抗体滴度。

图1显示，来自接种者的血清对B.1.1.529变体的中和程度远远低于分析的其他变体（B.1.1.7、B.1.351、B.1.617.2）。我们发现在接种同源BNT162b2或异源ChAdOx1初始免疫/BNT162b2加强免疫方案的个体中存在一些B.1.1.529交叉中和现象，但在接种同源ChAdOx1疫苗后没有。此外，虽然观察到对其他变体的交叉中和作用，但来自康复期个体的血清基本上不能中和B.1.1.529。

然而，所有来自超级免疫个体的血清，如果被感染并接种过一次或两次BNT162b2疫苗，或者接种过疫苗后被感染，都能够中和B.1.1.529，尽管程度低于B.1.617.2。

B.1.1.529是否会成为主导，并在未来造成重大的发病率和死亡率，将取决于其传播性和致病性。然而，尽管用BNT162b2进行第三次强化接种可能会提高对B.1.1.529的交叉中和抗体水平，但根据我们目前所得到的数据，我们强烈建议迅速开发新的、变异适应性强的疫苗。

支撑材料

方法

简而言之，血清在56°C下热灭活30分钟，随后在台式离心机上以8000转/离心5分钟。在含有2%胎牛血清（FCS）的完全培养基中，从1:16的稀释度开始，将血清一式两份样本稀释4倍。病毒（B.1.1.7：分离物C69.1，GISAID ID EPI_ISL_3277382；B.1.351：分离物C24.1，GISAID ID EPI_ISL_1123262；B.1.617.2：分离物SARS-CoV-2-hCoV-19/USA/NY-MSHSP-PV29995/2021，GISAID ID EPI_ISL_2290769；B.1 .1.529：隔离体E16.1，GISAID ID EPI_ISL_6902053），在没有血清的对照孔中加入约100-200个感染细胞，并将混合物在37℃下培养1小时。随后，血清/病毒混合物被转移到稳定过表达TMPRSS2和ACE2的汇合Vero细胞。感染两小时后，取出接种物并加入新鲜培养基。感染10小时后，用96%的乙醇固定细胞5分钟，随后用一个康复期病人的血清和Alexa Fluor Plus 488标记山羊抗人IgG二抗（Invitrogen, Thermo Fisher Scientific, Vienna, Austria）染色。使用ImmunoSpot S6 Ultra-V读数器和FluoroSpot软件（CTL Europe GmbH, Bonn, Germany）对平板进行分析。在GraphPad Prism 9.0.1（GraphPad Software, Inc., La Jolla, CA, USA）中使用非线性回归计算连续的50%中和效价。效价>1:16被认为是阳性。
